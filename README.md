# 💼 Personal Portfolio Website

Welcome to my **Developer Portfolio Website**! This is a modern, responsive, and animated web portfolio built with **React.js**, **Tailwind CSS**, **AOS (Animate on Scroll)**, and more. It showcases my work experience, projects, skills, and how to contact me.

---

## 🚀 Live Demo

🌐 [View Portfolio Live](https://superlative-cobbler-95f74a.netlify.app/)

---


## ✨ Features

- ⚡ Fast and responsive across all devices
- 🎨 Stylish design with smooth animations using AOS
- 💼 Dedicated sections for:
  - About Me
  - Skills
  - Work Experience
  - Projects
  - Contact Form
- 🌌 Dark-themed modern UI
- 🔗 Integrated social links
- 🛠 Reusable React components

---

## 🔧 Tech Stack

| Technology       | Description                        |
|------------------|------------------------------------|
| **React.js**     | Frontend library                   |
| **Tailwind CSS** | Utility-first CSS framework        |
| **AOS**          | Scroll animations                  |
| **React Icons**  | Icons used in sections             |
| **EmailJS / FormSubmit** | Contact form (optional)        |
